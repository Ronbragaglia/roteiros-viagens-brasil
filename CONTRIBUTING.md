# 🤝 Como Contribuir

Obrigado por querer contribuir! Este é um projeto colaborativo e toda ajuda é bem-vinda.

## 📋 Tipos de Contribuição

### 🗺️ Novo Roteiro de Cidade
1. Fork o repositório
2. Crie uma branch: `git checkout -b roteiro/nome-da-cidade`
3. Copie o template: `cp templates/roteiro-template.md roteiros/nome-da-cidade.md`
4. Preencha com informações **testadas pessoalmente** (ou de fontes confiáveis)
5. Abra um Pull Request

### ✏️ Correção/Atualização
- Restaurante fechou? Preço mudou? Abra uma Issue ou PR direto
- Atualize com a data da verificação

### 📸 Dicas Extras
- Dicas que só morador sabe
- Alertas de segurança atualizados
- Eventos sazonais

## ✅ Checklist do Roteiro

Antes de enviar, confirme:
- [ ] Dia a dia detalhado com horários
- [ ] Pelo menos 5 restaurantes com faixa de preço
- [ ] Pelo menos 3 opções de hospedagem (econômico a luxo)
- [ ] Como chegar (aeroporto, transfer, transporte local)
- [ ] Tabela de orçamento estimado
- [ ] Mínimo 5 dicas locais
- [ ] Alertas de segurança quando relevante

## 🚫 O que NÃO aceitar

- Conteúdo copiado de outros sites sem atribuição
- Publicidade disfarçada
- Informações não verificáveis
- Preços muito desatualizados (mais de 1 ano)

## 📝 Estilo

- Linguagem informal mas informativa
- Emojis são bem-vindos nos títulos
- Preços sempre em R$ (Real Brasileiro)
- Faixa de preço: $ (até R$50) | $$ (R$50-150) | $$$ (R$150-300) | $$$$ (R$300+)
- Use tabelas para organizar informações comparativas
