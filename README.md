## 🗂️ Estrutura dos Roteiros

Cada roteiro inclui:
- **Dia a dia** — itinerário detalhado com horários sugeridos
- **Onde comer** — restaurantes testados com faixa de preço ($, $$, $$$)
- **Onde ficar** — hospedagem por perfil (mochileiro, casal, família)
- **Como chegar** — aeroporto, rodoviária, transfers
- **Quanto custa** — orçamento estimado por dia/pessoa
- **Dicas locais** — o que só morador sabe

## 💰 Faixa de Orçamento
