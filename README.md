## 🗺️ Por que este guia é diferente?

| ❌ Guias tradicionais | ✅ Este guia |
|---|---|
| Informações genéricas de blog | Roteiro **dia a dia com horários** |
| "Visite o Cristo Redentor" | Qual horário ir, quanto custa, como evitar fila |
| Sem preços reais | **Preços atualizados em R$** com faixa ($-$$$$) |
| Foco em turista rico | **3 perfis de orçamento** (econômico, conforto, luxo) |
| Não fala o que evitar | **Dicas de segurança** reais e honestas |
| Conteúdo pago/patrocinado | **100% independente**, sem jabá |

## 📊 O que cada roteiro inclui

```
📅 Dia a dia          → Itinerário detalhado com horários (manhã/tarde/noite)
🍽️ Onde comer         → Restaurantes reais com prato recomendado e faixa de preço
🏨 Onde ficar         → Do hostel ao 5 estrelas, com diárias atualizadas
✈️ Como chegar        → Aeroporto, transfer, transporte local com preços
💰 Quanto custa       → Tabela comparativa: econômico vs conforto vs luxo
💡 Dicas locais       → O que só morador sabe (e guia turístico não conta)
⚠️ Segurança          → Alertas honestos sobre o que evitar
```

## 🤝 Como Contribuir

Viajou pra algum lugar e quer compartilhar? Perfeito!

1. **Fork** este repositório
2. **Copie** o [template](templates/roteiro-template.md)
3. **Escreva** seu roteiro com dicas reais
4. **Abra** um Pull Request

Veja o [guia completo de contribuição](CONTRIBUTING.md).

> Aceitamos: novos roteiros, correções de preço, restaurantes que abriram/fecharam, dicas de segurança atualizadas.
